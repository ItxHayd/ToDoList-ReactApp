React TypeScript To-Do List

A simple To-Do List app built with React and TypeScript, allowing users to:

Add tasks dynamically

Remove tasks

Toggle task states (like highlighting or marking as active)

This project demonstrates state management, event handling, and type-safe React code using TypeScript.
